EduLearnAI - Testing Links

This page provides direct links to test various features of the EduLearnAI application, particularly the password reset functionality.

Password Reset Testing

Test the password reset functionality using our custom test page:

  • Password Reset Test Page

Or test the API endpoints directly:

  • Request Password Reset: curl -X POST -H "Content-Type: application/json" -d '{"email": "test@example.com"}' http://localhost:5000/api/password-reset/request
  • Reset Password: curl -X POST -H "Content-Type: application/json" -d '{"token": "YOUR_TOKEN", "password": "NewPassword123", "confirmPassword": "NewPassword123"}' http://localhost:5000/api/password-reset/reset

API Endpoints

  • Get User (ID: 1)
  • Password Reset Routes (GET not supported)

Note: Most API endpoints require POST/PUT/DELETE methods and cannot be directly accessed via browser links.
